# Cetha AI - Autonomous SME Finance Assistance

Cetha AI adalah sistem fintech berbasis kecerdasan buatan yang dirancang sebagai *autonomous financial assistant* bagi UMKM di Surabaya. Sistem ini mentransformasi cara pelaku usaha mengelola bisnis melalui otomatisasi transaksi, pemantauan real-time, dan rekomendasi cerdas.

## 📋 Fitur Utama
* **Scan Nota Otomatis**: Membaca dan mengekstrak informasi dari nota fisik menggunakan teknologi OCR secara cepat dan akurat.
* **Kategorisasi Pengeluaran Otomatis**: Mengelompokkan transaksi ke dalam kategori seperti bahan baku atau operasional menggunakan algoritma AI.
* **Dashboard Keuangan**: Visualisasi data arus kas (total pengeluaran, distribusi kategori, riwayat) yang mudah dipahami.
* **Smart Insight**: Analisis otomatis untuk mendeteksi tren, dominasi kategori pengeluaran, hingga anomali transaksi dalam bentuk kalimat sederhana.
* **Saran Stok Musiman**: Rekomendasi jumlah dan jenis stok barang berdasarkan pola waktu dan data historis (contoh: menjelang Lebaran).

## 🛠️ Stack Teknologi
* **Frontend**: React / Next.js.
* **Backend**: Python (Flask / FastAPI).
* **AI Engine**: Google Vision API (OCR) & Gemini API (Processing).
* **Database**: Cloud Firestore / Firebase Storage.

## 🏗️ Arsitektur Sistem
Sistem dirancang dengan arsitektur berlapis yang memisahkan antara lapisan antarmuka pengguna (Frontend), lapisan logika AI dan backend, serta lapisan penyimpanan data untuk memastikan skalabilitas dan efisiensi.

## 📈 Dampak Inovasi
* **Efisiensi Waktu**: Menghilangkan proses pencatatan manual yang memakan waktu.
* **Akurasi Data**: Mengurangi human error melalui otomatisasi ekstraksi data.
* **Keputusan Tepat**: Membantu pengambilan keputusan bisnis melalui insight dan saran stok berbasis data.
